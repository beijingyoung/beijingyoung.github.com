Ultimate CSS3 Experiments

All demo use pure CSS, non-js!

Demo One : Fade Effect

#one {
	-webkit-transition: opacity 1.5s linear;
	-moz-transition: opacity 1.5s linear;
	-o-transition: opacity 1.5s linear;
	transition: opacity 1.5s linear;
}
#one:hover {
	opacity: 0;
	-moz-opacity: 0;
	filter:alpha(opacity=0);
}

Demo Two : Nudge Effect

#two {
	-webkit-transition-property: padding, color, background;
	-webkit-transition-duration: 2000ms, 900ms, 500ms;
}
#two:hover{
	padding: 1em 1em 1em 10em;
	background: #f2f2f2;
	color: green;
}

Demo Four : Bounce Effect

#four:hover {
	-webkit-animation-name: bounce;
	-webkit-animation-direction: alternate;
	-webkit-animation-duration: 3s;
	-webkit-animation-iteration-count: 6;
}
@-webkit-keyframes bounce {
	from {
		margin-left: 0;
	}
	to {
		margin-left: 360px;
	}
}
