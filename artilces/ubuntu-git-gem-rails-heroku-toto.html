Build personal blog on Heroku - Ubuntu + Ruby + Git + Gem + Rails + Heroku + toto

$ sudo apt-get install ruby
$ ruby -v

$ sudo apt-get install rails
$ rails -v

$ sudo apt-get install gem
$ sudo apt-get autoremove
$ sudo apt-get install rubygems1.9.1
$ gem -v
$ sudo gem install rubygems-update
$ gem environment
$ gedit ~/.bashrc #or vi ~/.bashrc
# Add the following code, on the end of the file :
export PATH=/var/lib/gems/1.8/bin:$PATH

$ echo $PATH
$ sudo gem install heroku
$ heroku version

$ sudo apt-get install git
$ git version

open heroku.com, register an account.
$ ssh-keygen -t rsa -C "webmaster@example.com"
# Replace webmaster@example.com to login into Heroku's email address.

$ git config --global user.name "Your Name"
$ git config --global user.email "webmaster@example.com"

$ git config --global heroku.user "Your Name"
$ git config --global heroku.token "1234567890abcdefg"
heroku.token is from your Heroku account API key.

$ sudo ssh git@heroku.com

$ git clone git://github.com/cloudhead/dorothy.git myblogname
# myblogname = myblogname.heroku.com
$ cd myblogname
$ heroku create myblogname
$ git push heroku master

Now this blog is successful build, please visit http://myblogname.heroku.com/.
View more detailed configuration please check toto website.

If your want to change your heroku domain.
$ git remote rm heroku
$ git remote add heroku git@heroku.com:domain.git


$ heroku keys:clear; sudo ssh-keygen -t rsa; heroku keys:add ~/.ssh/id_rsa.pub
